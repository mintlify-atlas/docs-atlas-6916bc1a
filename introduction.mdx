## Welcome to Simon Willison's Tools

This is an experimental collection of miscellaneous tools built mostly with the help of Large Language Models (LLMs). The collection demonstrates the power of prompt-driven development with very low stakes, showcasing what's possible when you combine human creativity with AI assistance.

All tools are hosted at [tools.simonwillison.net](https://tools.simonwillison.net/) and the complete source code is available on [GitHub](https://github.com/simonw/tools).

## Built with LLMs

Nearly every tool in this collection was created using AI assistance, particularly Claude. The [colophon](https://tools.simonwillison.net/colophon) provides commit messages and transcripts showing how each tool was built.

<Info>
Many tools used custom instructions [described in this blog post](https://simonwillison.net/2024/Dec/19/one-shot-python-tools/#custom-instructions) to streamline the development process.
</Info>

## Getting Started

<Steps>
  <Step title="Choose Your Tool Type">
    Browse **Browser Tools** for instant web-based utilities, or **Python Scripts** for command-line power tools.
  </Step>
  
  <Step title="Try a Tool">
    Browser tools work immediately in your browser. Python scripts can be run with `uv run` followed by the tool URL.
  </Step>
  
  <Step title="Explore the Source">
    Every tool links to its source code on GitHub. Learn how they work and adapt them for your needs.
  </Step>
</Steps>

## Popular Tools

Some of the most useful tools in the collection:

- **[OCR](https://tools.simonwillison.net/ocr)** - Recognize text from images and PDFs directly in your browser
- **[Social Media Cropper](https://tools.simonwillison.net/social-media-cropper)** - Crop images to 2×1 for social sharing
- **[Bluesky WebSocket Firehose](https://tools.simonwillison.net/bluesky-firehose)** - Watch real-time activity on Bluesky
- **[Claude Token Counter](https://tools.simonwillison.net/claude-token-counter)** - Count tokens for Claude prompts
- **[JSON Extractor](https://tools.simonwillison.net/python/json_extractor.py)** - Extract valid JSON objects from mixed text

<Note>
Check the [quickstart guide](/quickstart) to learn how to use both browser tools and Python scripts.
