A collection of client-side image and media processing tools that run entirely in your browser. From cropping social media images to converting between formats and working with SVG files, these tools handle visual content without uploading files to external servers.

## Image Conversion and Processing

<CardGroup cols={2}>
  <Card
    title="Image to JPEG"
    icon="file-image"
    href="https://tools.simonwillison.net/image-to-jpeg"
  >
    Convert PNG or WebP files to JPEG format directly in your browser
  </Card>
  
  <Card
    title="Image to SVG"
    icon="vector-square"
    href="https://tools.simonwillison.net/image-to-svg"
  >
    Trace bitmap images to SVG paths for scalable vector graphics
  </Card>
  
  <Card
    title="SVG to JPEG/PNG"
    icon="file-export"
    href="https://tools.simonwillison.net/svg-render"
  >
    Render SVG files to raster images in JPEG or PNG format
  </Card>
  
  <Card
    title="Image Resize and Quality Comparison"
    icon="magnifying-glass-chart"
    href="https://tools.simonwillison.net/image-resize-quality"
  >
    Compare different JPEG quality settings side-by-side
  </Card>
</CardGroup>
