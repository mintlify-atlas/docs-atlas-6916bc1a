Experiment with large language model APIs directly in your browser. These tools provide interfaces for testing prompts, analyzing responses, and exploring model capabilities with Claude, Gemini, OpenAI, and Chrome's built-in AI.

## Claude Tools

<CardGroup cols={2}>
  <Card
    title="Haiku"
    icon="camera"
    href="https://tools.simonwillison.net/haiku"
  >
    Generate haikus using Claude Haiku and your device camera
  </Card>
  
  <Card
    title="Claude Token Counter"
    icon="calculator"
    href="https://tools.simonwillison.net/claude-token-counter"
  >
    Count tokens in prompts to estimate Claude API usage and costs
  </Card>
</CardGroup>

<Note>
  The Haiku tool captures images from your camera and uses Claude's vision capabilities to compose original haiku poetry based on what it sees.
</Note>
