## GitHub API Tools

<CardGroup cols={2}>
  <Card
    title="GitHub API Write"
    icon="upload"
    href="https://tools.simonwillison.net/github-api-write"
  >
    Upload text content or images directly to GitHub repositories via the API
  </Card>
  
  <Card
    title="GitHub Issue Viewer"
    icon="circle-exclamation"
    href="https://tools.simonwillison.net/github-issue"
  >
    Fetch and display GitHub issues with all comments
  </Card>
  
  <Card
    title="GitHub Issue to Markdown"
    icon="markdown"
    href="https://tools.simonwillison.net/github-issue-to-markdown"
  >
    Convert an entire GitHub issue thread to Markdown format
  </Card>
</CardGroup>

<Note>
  The GitHub API Write tool requires a personal access token with appropriate repository permissions. The token is stored locally in your browser.
</Note>
