A suite of tools for working with Bluesky social network data. Monitor real-time activity, export threads, resolve user identifiers, and analyze post engagement—all running client-side in your browser.

## Real-Time Monitoring

<Card
  title="Bluesky WebSocket Firehose"
  icon="fire"
  href="https://tools.simonwillison.net/bluesky-firehose"
>
  Watch real-time Bluesky activity by connecting to the Jetstream WebSocket service. Send custom JSON messages to filter by collection type, specific user DIDs, or other parameters. All events are logged for debugging and inspection.
</Card>

<Note>
  Use Ctrl/Cmd + Enter to quickly send configuration updates to the WebSocket connection in the Firehose tool.
</Note>
