Simon Willison's Tools includes 28+ Python scripts that can be run directly from their URLs using `uv run`, without any installation required.

## How to Run Scripts

All scripts can be executed using the `uv run` command followed by the script's URL:

```bash
uv run https://tools.simonwillison.net/python/script_name.py [arguments]
```

The `uv` tool will automatically:
- Download the script
- Parse its dependencies from the inline script metadata
- Create an isolated environment
- Install required packages
- Execute the script

## Getting Help

Most scripts support the `--help` flag to see all available options:

```bash
uv run https://tools.simonwillison.net/python/script_name.py --help
```

## Requirements

You need [uv](https://docs.astral.sh/uv/) installed to run these scripts. Install it with:

```bash
# macOS/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```
