## Overview

The tools collection uses **Playwright** for browser automation testing with **pytest** as the test runner. This combination provides reliable, fast end-to-end testing of tools in real browsers.

<Note>
Tests run automatically on every push via GitHub Actions. All tests must pass before merging.
</Note>

### Shared Fixtures

The `tests/conftest.py:75` file provides reusable test fixtures:

```python
import socket
import subprocess
import sys
import time
import pytest


class StaticServer:
    """Manages a static file HTTP server with proper startup waiting."""

    def __init__(self, port):
        self.port = port
        self._process = None

    def start(self, directory='.'):
        """Start an HTTP server serving the specified directory."""
        self._process = subprocess.Popen(
            [sys.executable, '-m', 'http.server', str(self.port), 
             '--directory', str(directory)],
            stdout=subprocess.PIPE,
            stderr=subprocess.PIPE,
            text=True
        )
        
        # Wait for server to be ready (up to 5 seconds)
        import urllib.request
        for _ in range(50):
            time.sleep(0.1)
            try:
                urllib.request.urlopen(f'http://127.0.0.1:{self.port}/', timeout=1)
                return self
            except (urllib.error.URLError, ConnectionRefusedError):
                continue
        
        raise RuntimeError("Server did not become ready in time")

    def stop(self):
        """Stop the HTTP server if it's running."""
        if self._process is not None:
            self._process.terminate()
            self._process.wait(timeout=5)


@pytest.fixture
def unused_port():
    """Returns an unused port number on localhost."""
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    try:
        sock.bind(('127.0.0.1', 0))
        port = sock.getsockname()[1]
        return port
    finally:
        sock.close()


@pytest.fixture
def unused_port_server(unused_port):
    """
    Returns a StaticServer instance that can start an HTTP server on an unused port.
    The server automatically stops at the end of the test function.
    """
    server = StaticServer(unused_port)
    yield server
    server.stop()
```

### Basic Test Structure

```python
import pathlib
from playwright.sync_api import Page, expect

test_dir = pathlib.Path(__file__).parent.absolute()
root = test_dir.parent.absolute()


def test_initial_state(page: Page, unused_port_server):
    """Test that the page loads with correct initial state."""
    # Start the server
    unused_port_server.start(root)
    
    # Navigate to the tool
    page.goto(f"http://127.0.0.1:{unused_port_server.port}/ocr.html")
    
    # Assert initial state
    expect(page.locator("h1")).to_have_text(
        "OCR PDFs and images directly in your browser"
    )
    expect(page.locator("#dropzone")).to_have_text(
        "Drag and drop a PDF, JPG, PNG, or GIF file here or click to select a file"
    )
    expect(page.locator("#fullDocumentSection")).not_to_be_visible()
```

## Playwright API Reference

### Common Locators

```python
# By ID
page.locator("#element-id")

# By class
page.locator(".class-name")

# By tag
page.locator("button")

# By text
page.locator("text=Click me")

# By attribute
page.locator('[data-tab="paste"]')

# Combined selectors
page.locator("button.primary#submit")

# Nth element
page.locator(".item").nth(0)
page.locator(".item").first
page.locator(".item").last
```

### Common Assertions

```python
from playwright.sync_api import expect

# Text content
expect(page.locator("h1")).to_have_text("Expected Text")
expect(page.locator("p")).to_contain_text("partial")

# Visibility
expect(page.locator("#element")).to_be_visible()
expect(page.locator("#element")).not_to_be_visible()

# Values
expect(page.locator("input")).to_have_value("expected value")
expect(page.locator("select")).to_have_value("option1")

# Attributes
expect(page.locator("img")).to_have_attribute("src", "image.png")
expect(page.locator("a")).to_have_attribute("href", "https://example.com")

# Classes
expect(page.locator("button")).to_have_class("active")
expect(page.locator("button")).to_have_class("btn active")

# Count
expect(page.locator(".item")).to_have_count(3)

# Disabled state
expect(page.locator("button")).to_be_disabled()
expect(page.locator("button")).to_be_enabled()

# Page title
expect(page).to_have_title("Page Title")
```

### Common Actions

```python
# Click
page.locator("button").click()

# Fill input
page.locator("#input").fill("text")

# Clear input

# Type (with delays between keystrokes)
page.locator("#input").type("text", delay=100)

# Press keys
page.locator("#input").press("Enter")
page.locator("#input").press("Escape")

# Select dropdown

# Check/uncheck

page.locator("#checkbox").uncheck()

# Upload file

# Multiple files

```

### Waiting

```python
# Wait for element
page.wait_for_selector("#element")

# Wait for load state
page.wait_for_load_state("networkidle")

# Wait for timeout
page.wait_for_timeout(1000)  # milliseconds

# Wait for function
page.wait_for_function("() => document.querySelector('#element') !== null")

# Custom timeout in assertions
expect(page.locator("#element")).to_be_visible(timeout=10000)
```

```bash

pytest --browser chromium

pytest --browser firefox

pytest --browser webkit
```

### Run with Debugging

```bash

pytest -x

# Drop into debugger on failure
pytest --pdb

# Slow down execution
pytest --slowmo 1000  # 1 second delay between actions
```

### Run Tests Matching Pattern

```bash
# Run tests matching "upload"
pytest -k upload

# Run tests NOT matching "slow"
pytest -k "not slow"
```

## CI/CD Integration

Tests run automatically via GitHub Actions in `.github/workflows/test.yml`:

```yaml
name: Test
on:
  push:
  pull_request:

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v4
    
    - name: Set up Python 3.12
      uses: actions/setup-python@v5
      with:
        python-version: "3.12"
        cache: 'pip'
        cache-dependency-path: 'pyproject.toml'
    
    - name: Cache Playwright browsers
      uses: actions/cache@v4
      with:
        path: ~/.cache/ms-playwright/
        key: ${{ runner.os }}-browsers
    
    - name: Install dependencies
      run: |
        pip install -e .
        playwright install
    
    - name: Run tests
      run: pytest
```

## Test Best Practices

<CardGroup cols={2}>
  <Card title="Test User Flows" icon="route">
    Test complete user journeys, not just individual functions. How would a real user interact with your tool?
  </Card>
  
  <Card title="Test Edge Cases" icon="triangle-exclamation">
    Empty inputs, very long inputs, special characters, invalid data. Don't just test the happy path.
  </Card>
  
  <Card title="Use Descriptive Names" icon="tag">
    Test names should clearly describe what they're testing: `test_empty_input_shows_error` not `test_error`
  </Card>
  
  <Card title="Keep Tests Independent" icon="cube">
    Each test should be able to run independently. Don't rely on test execution order.
  </Card>
  
  <Card title="Handle Async" icon="clock">
    Use appropriate timeouts for async operations. Don't make tests flaky with hard-coded waits.
  </Card>
  
  <Card title="Clean Up" icon="broom">
    Use fixtures that clean up after themselves. The `unused_port_server` fixture stops the server automatically.
  </Card>
</CardGroup>

## Common Issues

<AccordionGroup>
  <Accordion title="Test times out waiting for element">
    **Causes:**
    - Element never appears
    - Selector is wrong
    - Async initialization not complete
    
    **Solutions:**
    ```python
    # Increase timeout
    expect(page.locator("#element")).to_be_visible(timeout=30000)
    
    # Wait for initialization first
    page.wait_for_load_state("networkidle")
    
    # Check selector
    print(page.content())  # See full HTML
    ```
  </Accordion>
  
  <Accordion title="Server fails to start">
    **Causes:**
    - Port already in use
    - Python not in PATH
    
    **Solutions:**
    ```python
    # The unused_port fixture automatically finds an available port
    # Make sure you're using it:
    def test_something(page: Page, unused_port_server):
        unused_port_server.start(root)
        # Use unused_port_server.port, not a hardcoded port
    ```
  </Accordion>
  
  <Accordion title="Tests pass locally but fail in CI">
    **Causes:**
    - Timing differences
    - Different browser versions
    - Missing test fixtures
    
    **Solutions:**
    - Use `expect()` assertions with timeouts instead of `wait_for_timeout()`
    - Ensure test fixtures are committed to git
    - Test with same browser version as CI
  </Accordion>
  
  <Accordion title="Flaky tests (sometimes pass, sometimes fail)">
    **Causes:**
    - Race conditions
    - Insufficient waits
    - Hardcoded timeouts
    
    **Solutions:**
    ```python
    # Bad - hardcoded wait
    page.wait_for_timeout(2000)
    
    # Good - wait for specific condition
    expect(page.locator("#result")).to_be_visible()
    
    # Wait for network to be idle
    page.wait_for_load_state("networkidle")
    ```
  </Accordion>
</AccordionGroup>
